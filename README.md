# 🧠 Memory Assistant

A personal memory diary enhanced with AI-powered semantic search.

## 🚀 About the Project
This project is designed to help users capture life's important moments and retrieve them instantly using intelligent search. No more struggling to remember exact dates or keywords—the AI understands the context and meaning of your entries.

## ✨ Key Features
* **Smart Search:** Find memories by meaning and emotion, not just keyword matching.
* **Streamlined UI:** A minimalist and intuitive interface for quick, distraction-free journaling.
* **AI Automation:** Leveraging modern LLMs to process and organize natural language.

## 🛠 Tech Stack
* **Frontend:** React / HTML5 / Tailwind CSS
* **Deployment:** [Vercel](https://memory-assistant-psi.vercel.app)
* **AI Integration:** OpenAI API / Vector Embeddings
