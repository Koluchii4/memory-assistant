# 🧠 Memory Assistant

A personal memory diary enhanced with AI-powered semantic search.

## 🚀 About the Project
This project is designed to help users capture life's important moments and retrieve them instantly using intelligent search. No more struggling to remember exact dates or keywords—the AI understands the context and meaning of your entries.
"Memory Assistant" is not just a diary. It's a time machine into your own life.
The core idea is simple: the earlier you start, the more valuable it becomes. Imagine being 40 and returning to a Friday afternoon when you were 20 — not through blurry general memories, but in vivid detail. The day you first said "I love you." The idea you once had and forgot. The person you used to be.
The app serves a wide range of people. For anyone who wants to remember their past. For astrologers — to cross-reference real life events with planetary positions and make precise forecasts. For people with dementia — so each new day is easier when yesterday is known. For families — to stay connected across time and distance.
We live in an age of information overload. Our brains simply cannot hold everything. Memory Assistant is your external memory — one that never forgets.

## ✨ Key Features
* **Smart Search:** Find memories by meaning and emotion, not just keyword matching.
* **Streamlined UI:** A minimalist and intuitive interface for quick, distraction-free journaling.
* **AI Automation:** Leveraging modern LLMs to process and organize natural language.

## 🛠 Tech Stack
* **Frontend:** React / HTML5 / Tailwind CSS
* **Deployment:** [Vercel](https://memory-assistant-psi.vercel.app)
* **AI Integration:** OpenAI API / Vector Embeddings
